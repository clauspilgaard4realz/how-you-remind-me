How You Remind Me

En privat reminder-app til én bruger, der bliver ved med at nagge dig, til opgaven er klaret. Dette dokument beskriver den valgte visuelle retning ("Nat"), designsystemet, hver skærm med specifikation, appens adfærd/logik, alle danske tekster og en foreslået datamodel — så det kan bygges direkte.

1 · Fundament

Princip: ét klart signal ad gangen. Mørk, rolig flade så den orange accent kun bruges til det der kræver handling (forfaldne, primære knapper, aktiv tilstand). Store, læsbare rækker; alt teknisk er skjult som standard. Mobil-først (iPhone).

Skærm 02 · hjerte i appen

Overblik · "I dag"

Løser den gamle apps problem med mange påmindelser. Rækkerne er prioriteret og grupperet, så det vigtigste altid står øverst.

  • Rækkefølge: (1) Forfaldne øverst i egen rød sektion m. antal · (2) grupperet efter tid: Senere i dag → I morgen → Senere/dato
  • Inden for en gruppe: sorteret på tidspunkt stigende
  • Rækkeanatomi: checkbox · titel · tidspunkt (højre) · meta-linje · evt. nag-chip
  • Gentagelse vises med ↻ + kadence ("↻ Dagligt", "↻ Månedligt · d. 1.")
  • Nag vises som separat chip ("Nagger hvert 15. min") — aldrig blandet sammen med gentagelsen
  • Forfalden række har handlinger inline: Klaret + Udsæt 15 min / 1 time / 1 dag / Vælg
  • Push-status er ét grønt chip; alt teknisk ligger bag Avanceret ›
  • + øverst → Ny opgave · tryk på række → detalje

Skærm 04

Ny / Redigér opgave

Samme formular til begge. To adskilte begreber — hold dem visuelt adskilt: Gentagelse (hvornår opgaven kommer igen) og Nag (rytmen mens den er åben).

  • Felter: titel · Gentagelse (Én gang / Dagligt / Ugentlig / Bestemte ugedage / Månedligt) · Nag (Hvert 15. min / Hver time / Kun én gang) · Startdato + tidspunkt (kvarter) · Slutdato (valgfri deadline)
  • Redigér gentagen opgave: "Ændringer gælder: Kun denne gang / Hele serien" står øverst. Ny opgave skjuler denne vælger (ingen serie endnu)
  • Slet-knap respekterer samme valg → "Slet hele serien" / "Slet denne"
  • Gem-knap hedder "Gem opgave" (ny) / "Gem ændringer" (redigér)

Skærm 05

Notifikation

Sådan ser en påmindelse ud på låseskærmen når den udløser. Personligheden ("nagger") lever her.

  • Titel = opgavens navn · body forklarer nag-rytmen
  • To handlinger direkte fra notifikationen: Klaret og Udsæt 15 min

Skærm 06

Tom tilstand

Når alt er klaret. Belønnende, ikke tomt. Stribet cirkel-placeholder + klokke, "Alt er klaret", og én tydelig CTA Ny opgave. Copy: "Ingen påmindelser lige nu. Tilføj en, så holder jeg øje og nagger dig, til den er klaret."

4 · Adfærd & logik

Nag (påmindelsesrytme)

Mens en opgave er åben, sender appen gentagne notifikationer med det valgte interval: Hvert 15. min, Hver time eller Kun én gang. Nagging stopper i det øjeblik opgaven markeres Klaret. Verbet skrives i nutid på dansk: "nagger".

Gentagelse vs. instanser

En gentagen opgave (Dagligt/Ugentlig/Ugedage/Månedligt) er en serie. Hver forekomst er en instans med egen status. Klaret lukker kun dagens instans; næste dukker op ved næste forekomst. I overblikket markeres serier med ↻ + kadence.

Redigér & slet

Ved redigering/sletning af en instans i en serie spørges: Kun denne gang eller Hele serien. "Kun denne gang" laver en undtagelse på instansen; "Hele serien" ændrer skabelonen fremadrettet. Engangsopgaver springer valget over.

Forfaldne & snooze

Forfalden = planlagt tid passeret og status åben. Vises øverst i rød sektion med "X min/timer forsinket". Handlinger: Klaret + Udsæt 15 min / 1 time / 1 dag / Vælg (custom dato+tid). Udsæt flytter kun næste påmindelse — den rører ikke serien.

Push-status (skjult bag "Avanceret")

Standard er ét grønt Push aktiv-chip. Den tekniske boks fra den gamle app flyttes ind under Avanceret › og er foldet sammen som udgangspunkt:

Notification.permission — default/granted/denied
PushManager permission — prompt/granted
Subscription — aktiv/mangler
Service worker — aktiv
PWA standalone — ja/nej
Seneste dispatch — tidsstempel
Åbne uden enhed — antal
E-mail backup — konfigureret?

Hvis PWA ikke er installeret på iPhone, vis hint: "Åbn i Safari → Del → Føj til hjemmeskærm → åbn appen derfra for at aktivere push."

6 · Foreslået datamodel

Til inspiration — tilpas til din stack. En opgave (serie-skabelon) + beregnede instanser:

Task { id: string title: string recurrence: "once" | "daily" | "weekly" | "weekdays" | "monthly" weekdays?: number[] // ved "weekdays", 0–6 monthDay?: number // ved "monthly", 1–31 time: "HH:MM" // kvarter-præcision startDate: date endDate?: date // valgfri deadline nag: "q15" | "hourly" | "once" // rytme mens åben } TaskInstance { taskId: string date: date status: "open" | "done" snoozedUntil?: timestamp // sat af Udsæt nextFireAt: timestamp // beregnet: max(planlagt, snoozedUntil) overrides?: Partial<Task> // ved "kun denne gang" }

Sortering i overblik: forfaldne (nextFireAt < nu && open) først, derefter grupperet på date og sorteret på time.
